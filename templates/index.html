SETUP GUIDE

How to install, run, and populate the NeuroVision database.

System Requirements

Prerequisites

  • Python 3.8+ installed
  • pip package manager
  • CMake (required for dlib/face_recognition)
  • Webcam (optional, for live capture)
Installation Steps
1
Install CMake & dlib dependencies
sudo apt-get install cmake build-essential # Windows: install CMake from cmake.org
2
Install Python dependencies
pip install flask face_recognition Pillow numpy
3
Run the application
python app.py

Open browser → http://localhost:5000

How to Upload Data to Database

Method 1 — Web UI (Recommended)

  1. Go to the Register tab
  2. Fill in Name, Age, Gender, Course
  3. Capture photo via webcam or upload an image
  4. Click Register Person
  5. Face encoding is extracted and saved automatically

Method 2 — Bulk Import via API (curl)

POST to /api/register with JSON body:

curl -X POST http://localhost:5000/api/register \ -H "Content-Type: application/json" \ -d '{ "name": "Priya Nair", "age": 21, "gender": "Female", "course": "M.Sc Data Science", "image": "data:image/jpeg;base64,..." }'

Method 3 — Python Script (Batch Upload)

import requests, base64 def register(name, age, gender, course, img_path): with open(img_path, 'rb') as f: b64 = base64.b64encode(f.read()).decode() requests.post('http://localhost:5000/api/register', json={ 'name': name, 'age': age, 'gender': gender, 'course': course, 'image': 'data:image/jpeg;base64,' + b64 }) register("Rahul Kumar", 19, "Male", "B.Tech CSE", "photos/rahul.jpg")
Database Location

SQLite File

The database is stored at:

neurovision/ ├── database/ │ └── neurovision.db ← SQLite DB ├── app.py └── templates/ └── index.html

Use DB Browser for SQLite (free GUI tool) to inspect or manually edit records.
